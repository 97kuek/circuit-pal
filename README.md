# Circuit-Pal ⚡ 電子工作ツールキット

電子工作愛好家、メイカー、エンジニアのための、シンプルで実用的な計算＆リファレンスツール集です。
直感的なモダンUIで、日々の「ちょっとした計算」をスムーズにします。
**PWA (Progressive Web App)** に対応しており、インストールしてオフラインでも使用可能です。

## ✨ 特徴

- 🌟 **お気に入り機能** — よく使うツールをトップに固定
- 📜 **計算履歴** — 過去の計算結果を自動保存・復元
- ⌨️ **ショートカット** — Enterで計算、Escでクリアなどキーボード操作に対応
- 🌙 **ダークモード対応** — システム設定連動＋手動切替
- � **PWA対応** — アプリとしてインストール・オフライン動作
- �📋 **コピー機能** — 全計算結果をワンクリックでクリップボードにコピー
- 🧭 **統合ナビゲーション** — ヘッダーからすべてのツールに直接アクセス

## 🔧 機能一覧

### 抵抗ツールキット
- **カラーコード読取** — 4本・5本帯のカラーコードから抵抗値を算出
- **逆引き検索** — 抵抗値（例: 4.7kΩ）からE系列に基づいてカラーコードを逆算
- **SMDコードデコーダー** — 3桁/4桁/EIA-96のSMD抵抗コードを解読
- **LED抵抗計算** — 電源電圧・LED Vf・電流から直列抵抗値を計算・選定

### 単位変換 (Unit Converter)
- **抵抗・容量・インダクタンス・周波数** — 接頭辞（p, n, µ, m, k, M, G）間の相互変換
- リアルタイム更新

### RCフィルタ / 時定数計算
- **時定数 (τ)** と **カットオフ周波数 (fc)** をRとCから計算
- 抵抗・容量の単位を自由に選択可能

### オームの法則計算
- 電圧(V)・電流(I)・抵抗(R)・電力(P)のうち2つを入力で残りを自動計算
- 矛盾する値のエラー検知

### 555タイマー計算
- **非安定モード（発振）** — R1・R2・Cから周波数・デューティー比を算出
- **単安定モード** — R・Cからパルス幅を算出
- 回路図・波形イメージ付き

### 分圧回路計算
- **出力電圧計算** — Vin・R1・R2から出力電圧を算出
- **R2選定** — 目標出力電圧からR2の最適値を計算（E24系列対応）

### パターン幅計算（IPC-2221準拠）
- 電流・銅箔厚・温度上昇から必要なパターン幅を計算
- 内層/外層対応

### コンデンサ容量デコーダー
- 3桁コード（例: 104）から静電容量（pF / nF / µF）に変換
- よく使うコードの早見表付き

### 部品在庫管理
- 部品の種類・値・数量・保管場所をローカル管理
- **インライン数量変更** — リスト上で直接数量を増減
- **CSVインポート/エクスポート** — データのバックアップと移行
